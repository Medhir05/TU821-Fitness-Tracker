void WeeklyWorkout::displayWeeklyWorkout ( ) const

Displays the entire weekly workout plan.

Displays the weekly workout plan.

Outputs the workout plan for all seven days to the console. If a day has no exercises scheduled, it indicates that the day is empty.

Note
This function is intended for quick reference and does not save the plan to a file.
